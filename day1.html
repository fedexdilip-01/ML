🌿 Iris dataset: complete hands‑on example
📂 1. Load dataset & initial inspection (Pandas + NumPy)
🎯 WHAT

Load the classic Iris dataset (150 samples, 4 numeric features, 3 species). Compute basic descriptive statistics and array-level operations.

🧠 WHY

Pandas provides high-level summary (describe(), mean()). NumPy enables fast mathematical operations on underlying arrays.

📍 WHERE in ML pipeline

EDA (Exploratory Data Analysis) – understand feature distributions, class balance, and preliminary scaling needs.

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris

# Load iris as pandas DataFrame
iris = load_iris()
df = pd.DataFrame(data=iris.data, columns=iris.feature_names)
df['species'] = iris.target_names[iris.target]

# Basic stats using pandas & numpy
print("🔹 First 5 rows:")
print(df.head())

print("\n🔹 Descriptive statistics (mean, std, min, max etc):")
print(df.describe())

# Compute mean & std using NumPy on feature columns
feature_cols = iris.feature_names
mean_np = np.mean(df[feature_cols], axis=0)
std_np = np.std(df[feature_cols], axis=0, ddof=0)  # population std

print("\n🔹 NumPy computed mean per feature:")
for name, m in zip(feature_cols, mean_np):
    print(f"   {name:20s} : {m:.4f}")
print("\n🔹 NumPy computed std deviation:")
for name, s in zip(feature_cols, std_np):
    print(f"   {name:20s} : {s:.4f}")
📌 Sample output snippet:
sepal length (cm) mean = 5.8433, std ≈ 0.8281 ; petal width (cm) mean = 1.1993, std ≈ 0.7632
Pandas .describe() also shows quartiles, count, min/max — perfect for initial sanity checks.
📐 2. Why compute mean & std? (Normalization / Z-score context)
⚡ What we compute

Mean → central tendency; Standard deviation → spread of data. For each numeric feature, we calculate both.

💡 Why it matters

Many ML algorithms (SVM, KNN, neural nets) assume features on similar scale. Mean & std are used for standardization (Z-score). Outlier detection also leverages these.

⚙️ Where applied

Preprocessing step: sklearn.preprocessing.StandardScaler uses (x - mean)/std. Also used to understand feature dominance (e.g., petal length variability).

# Quick standardization manually using computed mean & std
features = df[feature_cols].values  # numpy array
mean_vals = np.mean(features, axis=0)
std_vals = np.std(features, axis=0)
features_standardized = (features - mean_vals) / std_vals
print("Standardized first 3 samples (first 2 features):\n", features_standardized[:3, :2])
📈 3. Scatter Matrix: multi‑dimensional visualization
👁️ What is it?

Grid of scatter plots for each pair of features. Diagonal shows histogram (or KDE) of each variable. Colors represent species.

🎯 Why use scatter matrix?

Reveals correlations, clusters, separability between classes. Iris is famous: setosa is linearly separable, versicolor & virginica show overlap.

🛠️ Where in ML workflow

Feature selection (detect redundant features), identify non‑linear patterns, guide dimensionality reduction (PCA).

from pandas.plotting import scatter_matrix

# Scatter matrix with color by species
fig, ax = plt.subplots(figsize=(12, 10))
scatter_matrix(df, alpha=0.7, figsize=(12, 10), diagonal='hist',
               c=df['species'].astype('category').cat.codes, cmap='viridis',
               marker='o', hist_kwds={'bins': 20}, s=40)
plt.suptitle('Iris Dataset - Scatter Matrix (colored by species)', 
             y=0.93, fontsize=14)
plt.gcf().tight_layout()
# plt.savefig('iris_scatter_matrix.png')  # optional
plt.show()
✅ Interpretation example:
• Sepal length vs petal width → positive correlation (points trend upward).
• Petal length vs petal width → strong linear trend, useful for classification.
• Histogram diagonals: petal width/length show bimodality because of species separation.

📝 Complete reproducible script (Iris → stats → scatter matrix)
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris
from pandas.plotting import scatter_matrix

# ---------- 1. Load ----------
iris_data = load_iris()
df = pd.DataFrame(iris_data.data, columns=iris_data.feature_names)
df['species'] = iris_data.target_names[iris_data.target]

# ---------- 2. Compute mean & std using NumPy ----------
feature_names = iris_data.feature_names
features_np = df[feature_names].values
mean_array = np.mean(features_np, axis=0)
std_array = np.std(features_np, axis=0)

print("===== Iris Feature Statistics =====")
for i, col in enumerate(feature_names):
    print(f"{col:20s} → mean = {mean_array[i]:.4f}, std = {std_array[i]:.4f}")

# ---------- 3. Pandas describe for comparison ----------
print("\n===== Pandas describe (quick overview) =====")
print(df.describe())

# ---------- 4. Build a nice scatter matrix ----------
fig = plt.figure(figsize=(11, 9))
scatter_matrix(df, alpha=0.7, figsize=(11, 9), diagonal='hist',
               c=df['species'].astype('category').cat.codes, cmap='plasma',
               marker='.', s=55, hist_kwds={'bins': 18})
plt.suptitle('Iris: Scatter Matrix – Pairwise relationships & species patterns', 
             fontsize=13, y=0.94)
plt.tight_layout()
plt.show()

# Bonus: group stats
print("\n===== Per species mean ± std (Pandas) =====")
print(df.groupby('species')[feature_names].agg(['mean', 'std']).round(3))
