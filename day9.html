🔗
Fully Connected

Dense Layers

What? Each neuron connects to all neurons in previous layer. Computes y = activation(W·x + b).

Why? Universal approximators – can learn any function with enough neurons.

Where? MLPs, final classification layers, feature extraction.

⚡
Non-linearity

Activation Functions

What? ReLU, Sigmoid, Tanh, Softmax. Introduce non-linearity.

Why? Without them, network collapses to linear model.

Where? Hidden layers (ReLU), output (Sigmoid/Softmax).

🔄
Learning Algorithm

Backpropagation

What? Chain rule to compute gradients, update weights via gradient descent.

Why? Enables end-to-end learning of deep networks.

Where? Every neural network training loop.

🔗 1. Dense (Fully Connected) Layers
📐 Mathematical operation: output = activation(W · input + b)
📊 Weight matrix W

Shape (units_in, units_out). Learned during training.

🧠 Bias vector b

Allows shifting activation, improves flexibility.

⚙️ Why dense?

Maximum representational power, but computationally expensive.

📌 Layer output shape: (batch_size, units_out)

⚡ 2. Activation Functions – Why They Matter
📈 ReLU, Sigmoid, Tanh, Softmax comparison
ReLU (Rectified Linear Unit)

f(x) = max(0, x). Most common for hidden layers. Avoids vanishing gradient.

Sigmoid

f(x) = 1/(1+e⁻ˣ). Outputs (0,1) – good for binary classification output.

Softmax

Converts logits to probabilities summing to 1 – multi-class output.

ReLU (fast, non-saturating), Sigmoid (probabilities), Tanh (zero-centered), Softmax (multi-class)

import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-5, 5, 100)
relu = np.maximum(0, x)
sigmoid = 1 / (1 + np.exp(-x))
tanh = np.tanh(x)

plt.figure(figsize=(10,4))
plt.subplot(1,3,1); plt.plot(x, relu, 'r-'); plt.title('ReLU'); plt.grid(alpha=0.3)
plt.subplot(1,3,2); plt.plot(x, sigmoid, 'g-'); plt.title('Sigmoid'); plt.grid(alpha=0.3)
plt.subplot(1,3,3); plt.plot(x, tanh, 'm-'); plt.title('Tanh'); plt.grid(alpha=0.3)
plt.tight_layout()
plt.show()
🔄 3. Backpropagation – The Learning Engine
🎯 Chain rule + Gradient Descent = Weight Updates
📐 Forward pass

Compute predictions and loss.

🔄 Backward pass

Compute gradients of loss w.r.t each weight using chain rule.

⚡ Update rule

w = w - learning_rate * gradient.

📌 ∂L/∂w = (∂L/∂ŷ) · (∂ŷ/∂z) · (∂z/∂w) where z = w·x + b
# Conceptual backpropagation step (simplified)
import numpy as np

def simple_nn_forward_backward(x, y_true, w1, w2, lr=0.01):
    # Hidden layer
    h = np.maximum(0, np.dot(x, w1))  # ReLU
    # Output layer (sigmoid)
    z = np.dot(h, w2)
    y_pred = 1 / (1 + np.exp(-z))
    # Loss (binary cross-entropy)
    loss = - (y_true * np.log(y_pred + 1e-8) + (1 - y_true) * np.log(1 - y_pred + 1e-8))
    # Gradients (simplified)
    d_loss = y_pred - y_true
    d_w2 = np.dot(h.T, d_loss)
    d_h = np.dot(d_loss, w2.T)
    d_w1 = np.dot(x.T, d_h * (h > 0))  # ReLU derivative
    # Update
    w1 -= lr * d_w1
    w2 -= lr * d_w2
    return loss.mean(), w1, w2

print("Backpropagation updates weights to minimize loss")
💡 Key insight: Backpropagation computes how much each weight contributed to the error, then adjusts weights to reduce future errors.
👕 Fashion MNIST Classification – 2-Layer MLP
🏷️ 10 clothing categories: T-shirt, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle boot
📊 Dataset

60k train, 10k test. 28x28 grayscale images → 784 features.

🎯 Architecture

Input (784) → Dense(128, ReLU) → Dense(10, Softmax).

⚙️ Training

Sparse Categorical Crossentropy loss, Adam optimizer, 10 epochs.

import tensorflow as tf
from tensorflow import keras
import numpy as np
import matplotlib.pyplot as plt

# Load Fashion MNIST
fashion_mnist = keras.datasets.fashion_mnist
(X_train, y_train), (X_test, y_test) = fashion_mnist.load_data()

# Normalize pixel values to [0,1]
X_train = X_train.astype('float32') / 255.0
X_test = X_test.astype('float32') / 255.0

# Flatten images (28x28 → 784)
X_train_flat = X_train.reshape(-1, 784)
X_test_flat = X_test.reshape(-1, 784)

print(f"Train shape: {X_train_flat.shape}, Test shape: {X_test_flat.shape}")

# Build 2-layer MLP
model = keras.Sequential([
    keras.layers.Dense(128, activation='relu', input_shape=(784,), name='hidden_layer'),
    keras.layers.Dense(10, activation='softmax', name='output_layer')
])

# Compile model
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Model summary
model.summary()

# Train
history = model.fit(X_train_flat, y_train,
                    epochs=10,
                    batch_size=32,
                    validation_split=0.1,
                    verbose=1)

# Evaluate
test_loss, test_acc = model.evaluate(X_test_flat, y_test, verbose=0)
print(f"\nTest accuracy: {test_acc:.4f}")

# Plot training history
plt.figure(figsize=(12,4))
plt.subplot(1,2,1)
plt.plot(history.history['accuracy'], label='Train Acc')
plt.plot(history.history['val_accuracy'], label='Val Acc')
plt.xlabel('Epoch'); plt.ylabel('Accuracy'); plt.legend(); plt.title('Training Progress')
plt.subplot(1,2,2)
plt.plot(history.history['loss'], label='Train Loss')
plt.plot(history.history['val_loss'], label='Val Loss')
plt.xlabel('Epoch'); plt.ylabel('Loss'); plt.legend(); plt.title('Loss Curves')
plt.tight_layout()
plt.show()
📊 Expected results: 2-layer MLP achieves ~88-90% test accuracy on Fashion MNIST. Deeper networks can reach 92-93%.

🎛️ Key Hyperparameters & Best Practices
📊 Architecture

Number of layers, neurons per layer. Start small, increase if underfitting.

⚙️ Training params

Learning rate (0.001 typical), batch size (32/64), epochs (monitor validation loss).

🛡️ Regularization

Dropout, Batch Normalization, L2 regularization – prevent overfitting.

# Adding regularization to improve generalization
regularized_model = keras.Sequential([
    keras.layers.Dense(128, activation='relu', kernel_regularizer='l2', input_shape=(784,)),
    keras.layers.Dropout(0.3),
    keras.layers.Dense(64, activation='relu', kernel_regularizer='l2'),
    keras.layers.Dropout(0.3),
    keras.layers.Dense(10, activation='softmax')
])
regularized_model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])
regularized_model.fit(X_train_flat, y_train, epochs=15, validation_split=0.1, verbose=0)
test_acc_reg = regularized_model.evaluate(X_test_flat, y_test, verbose=0)[1]
print(f"Regularized model test accuracy: {test_acc_reg:.4f} (vs {test_acc:.4f} baseline)")
