🔍 Day 7 – Unsupervised Learning: Clustering & PCA

K-Means, Elbow Method, Dimensionality Reduction | Mall Customer Segmentation with Visual Clusters
🎯
Clustering

K-Means

What? Partitions data into K clusters by minimizing within-cluster sum of squares (inertia).

Why? Fast, scalable, works well for spherical clusters.

Where? Customer segmentation, image compression, anomaly detection.

📊
Optimal K

Elbow Method

What? Plot inertia vs K; choose K at "elbow" where decrease slows.

Why? Balance between underfitting & overfitting clusters.

Where? Determining number of clusters before segmentation.

📐
Dim Reduction

PCA

What? Projects data onto principal components (max variance directions).

Why? Visualize high-dim data, reduce noise, speed up algorithms.

Where? Data exploration, feature extraction, preprocessing for clustering.

🎯 1. K-Means Clustering – How it Works
🔄 Iterative Expectation-Maximization algorithm
📐 Algorithm steps

1) Initialize K centroids. 2) Assign points to nearest centroid. 3) Update centroids as mean of assigned points. Repeat until convergence.

⚡ Inertia (WCSS)

Sum of squared distances from points to their cluster centroid. Minimized by K-Means.

⚠️ Limitations

Sensitive to initialization, assumes spherical clusters, requires K known. Use K-Means++ (default in sklearn) to improve.

📌 Inertia = Σᵢ₌₁ᴺ ||xᵢ - μₖ||² where μₖ is centroid of assigned cluster
from sklearn.cluster import KMeans
import numpy as np

# Simple synthetic demo
X_demo = np.random.rand(100, 2) * 10
kmeans_demo = KMeans(n_clusters=3, random_state=42, n_init=10)
kmeans_demo.fit(X_demo)
print(f"Cluster centers:\n{kmeans_demo.cluster_centers_}")
print(f"Inertia (WCSS): {kmeans_demo.inertia_:.2f}")
📈 2. Elbow Method – Finding Optimal K
🎯 Balance between compactness and number of clusters
📊 What to look for

Compute inertia for K=1 to 10. Elbow point is where marginal gain drops dramatically.

🧠 Why not just maximize clusters?

More clusters always reduce inertia but overfit. Elbow balances interpretability.

🔍 Alternatives

Silhouette score, Gap statistic. Elbow is quick & visual.

import matplotlib.pyplot as plt

# Calculate inertia for K=1..10
inertias = []
K_range = range(1, 11)
for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    kmeans.fit(X_demo)
    inertias.append(kmeans.inertia_)

plt.figure(figsize=(8,4))
plt.plot(K_range, inertias, 'bo-', linewidth=2, markersize=8)
plt.xlabel('Number of clusters (K)')
plt.ylabel('Inertia (WCSS)')
plt.title('Elbow Method for Optimal K')
plt.grid(alpha=0.3)
plt.show()
💡 Elbow interpretation: Look for "bend" where inertia starts decreasing linearly. For mall customers, elbow is often at K=4 or K=5.
📐 3. PCA – Visualizing High-Dimensional Data
🧮 Principal Component Analysis explained
⚙️ What PCA does

Finds directions (principal components) that maximize variance. Projects data onto top 2-3 components for visualization.

🎯 Why use PCA before clustering?

Reduces noise, speeds up K-Means, enables 2D scatter of clusters.

📊 Explained variance ratio

Proportion of total variance captured by each component. Aim for 70-90% cumulative.

from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler

# Generate sample 5D data, reduce to 2D
np.random.seed(42)
X_highdim = np.random.randn(200, 5)
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X_highdim)
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X_scaled)
print(f"Explained variance ratio: {pca.explained_variance_ratio_}")
print(f"Cumulative: {sum(pca.explained_variance_ratio_):.3f}")
🛍️ Customer Segmentation on Mall Dataset
🏬 Real-world: Segment shoppers by Annual Income & Spending Score
📊 Dataset

Mall Customer Data: CustomerID, Gender, Age, Annual Income (k$), Spending Score (1-100).

🎯 Business goal

Identify customer segments for targeted marketing (e.g., high income low spenders, young spenders).

🧠 Why unsupervised

No labels – discover natural groupings based on behavior.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA

# Load or create Mall dataset (simulated real-like data)
np.random.seed(42)
n_customers = 200
annual_income = np.random.normal(60, 20, n_customers).clip(15, 140)
spending_score = np.random.normal(50, 25, n_customers).clip(1, 100)
# Create some structure: higher income sometimes correlated with spending
spending_score = spending_score + 0.2 * (annual_income - 60) + np.random.randn(n_customers)*12
spending_score = spending_score.clip(1, 100)
age = np.random.normal(40, 15, n_customers).clip(18, 70)

df_mall = pd.DataFrame({
    'Annual Income (k$)': annual_income.astype(int),
    'Spending Score (1-100)': spending_score.astype(int),
    'Age': age.astype(int)
})
print("Mall Dataset (first 5 rows):")
print(df_mall.head())

# Feature selection for clustering (using Income & Spending Score, plus Age optional)
features = ['Annual Income (k$)', 'Spending Score (1-100)']
X_mall = df_mall[features].values

# Scale features (important for K-Means distance-based)
scaler = StandardScaler()
X_scaled_mall = scaler.fit_transform(X_mall)

# Determine optimal K using elbow method
inertias_mall = []
K_range = range(1, 11)
for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    kmeans.fit(X_scaled_mall)
    inertias_mall.append(kmeans.inertia_)

plt.figure(figsize=(8,4))
plt.plot(K_range, inertias_mall, 'bo-', linewidth=2)
plt.xlabel('Number of clusters (K)')
plt.ylabel('Inertia')
plt.title('Elbow Method for Mall Customers (Income vs Spending)')
plt.grid(alpha=0.3)
plt.show()

# Choose K=5 (common elbow in mall dataset)
optimal_k = 5
kmeans_final = KMeans(n_clusters=optimal_k, random_state=42, n_init=10)
clusters = kmeans_final.fit_predict(X_scaled_mall)
df_mall['Cluster'] = clusters

print(f"\nCluster centers (original scale):")
centers_original = scaler.inverse_transform(kmeans_final.cluster_centers_)
for i, center in enumerate(centers_original):
    print(f"Cluster {i}: Income=${center[0]:.0f}k, Spending={center[1]:.0f}")
📊 Elbow result: Typically K=5 is optimal for mall data → segments: high income high spend, high income low spend, low income high spend, low income low spend, average.
🖼️ Visualizing Customer Segments (2D Cluster Plot)
📊 Cluster visualization – Annual Income vs Spending Score
📈 What to observe

Each color = one segment. Centers marked with X. Clear separation of customer types.

🎯 Business interpretation

Target high-income low-spend with premium offers; high-income high-spend with loyalty programs.

⬆️ Visualization of K-Means clusters (K=5) on Mall dataset – each color = distinct customer segment.

# Actual code to generate the cluster scatter plot
plt.figure(figsize=(10,6))
colors = ['#e63946', '#1e6091', '#2e9c4a', '#f4a261', '#9c27b0']
for i in range(optimal_k):
    plt.scatter(X_mall[clusters == i, 0], X_mall[clusters == i, 1], 
                c=colors[i], label=f'Cluster {i}', alpha=0.6, s=50)
plt.scatter(centers_original[:,0], centers_original[:,1], 
            marker='X', s=200, c='black', edgecolors='white', linewidth=2, label='Centroids')
plt.xlabel('Annual Income (k$)')
plt.ylabel('Spending Score (1-100)')
plt.title('Customer Segmentation using K-Means (K=5)')
plt.legend()
plt.grid(alpha=0.3)
plt.show()
📐 PCA + Clustering on Multi-Feature Data
🔍 Using PCA to reduce dimensions before clustering (Age + Income + Spending)
📊 Multi-dimensional

Include Age, Income, Spending. PCA reduces to 2D for visualization while preserving variance.

⚙️ Why PCA first

Reduces noise, faster clustering, visualizable clusters.

# Using 3 features: Age, Income, Spending
features_pca = ['Annual Income (k$)', 'Spending Score (1-100)', 'Age']
X_multi = df_mall[features_pca].values
X_multi_scaled = StandardScaler().fit_transform(X_multi)

# Apply PCA
pca = PCA(n_components=2)
X_pca_mall = pca.fit_transform(X_multi_scaled)
print(f"Explained variance by 2 components: {pca.explained_variance_ratio_.sum():.3f}")

# K-Means on PCA-reduced data
kmeans_pca = KMeans(n_clusters=5, random_state=42, n_init=10)
clusters_pca = kmeans_pca.fit_predict(X_pca_mall)

plt.figure(figsize=(8,6))
scatter = plt.scatter(X_pca_mall[:,0], X_pca_mall[:,1], c=clusters_pca, cmap='viridis', alpha=0.6)
plt.xlabel('First Principal Component')
plt.ylabel('Second Principal Component')
plt.title('Customer Segments after PCA (Income + Spending + Age)')
plt.colorbar(scatter)
plt.show()
