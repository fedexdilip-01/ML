🖼️ Day 10 – Convolutional Neural Networks (CNNs) for Image Data

Convolution, Pooling, Data Augmentation | CIFAR-10 Classifier with Conv2D Layers
🔍
Feature Extraction

Convolution

What? Slides filters (kernels) over image to detect patterns (edges, textures).

Why? Preserves spatial structure, shares weights, fewer parameters than dense.

Where? Image classification, object detection, segmentation.

📏
Downsampling

Pooling

What? Max/Average pooling reduces spatial dimensions.

Why? Reduces computation, provides translation invariance, prevents overfitting.

Where? Between conv layers, before dense layers.

🔄
Regularization

Data Augmentation

What? Apply random transformations (rotation, flip, zoom) during training.

Why? Creates more varied data, reduces overfitting, improves generalization.

Where? Limited datasets, medical imaging, real-world deployment.

🔍 1. Convolution – How CNNs See Images
📐 Mathematical operation: (I * K)(x,y) = ΣᵢΣⱼ I(x+i, y+j) · K(i,j)
📊 Filter/Kernel

Small matrix (e.g., 3x3) that slides across image. Detects features like edges, corners.

🧠 Feature maps

Each filter produces one feature map. Multiple filters = multiple feature detectors.

⚙️ Parameters

Conv layer has far fewer params than dense (e.g., 3x3x3x32 = 864 vs millions).

Filter (kernel) slides across image → element-wise multiplication + sum → feature map

import tensorflow as tf
from tensorflow import keras

# Conv2D layer example
conv_layer = keras.layers.Conv2D(filters=32, kernel_size=3, activation='relu', input_shape=(32,32,3))
# Forward pass on dummy CIFAR-like image
dummy_image = tf.random.normal((1, 32, 32, 3))
output = conv_layer(dummy_image)
print(f"Input shape: {dummy_image.shape} -> Output shape: {output.shape}")  # (1, 30, 30, 32)
📏 2. Pooling – Downsampling for Efficiency
⬇️ MaxPooling2D: Takes maximum value in each window
📊 MaxPooling

2x2 window with stride 2 reduces dimensions by 50%. Keeps strongest features.

🧠 Why pooling?

Reduces parameters, controls overfitting, provides small translation invariance.

⚙️ Alternative

AveragePooling (smoother), Strided Convolution (learnable downsampling).

MaxPooling takes maximum in each 2x2 window → reduces size, keeps dominant features

import tensorflow as tf

# MaxPooling example
pool_layer = keras.layers.MaxPooling2D(pool_size=(2,2), strides=2)
input_feature = tf.random.normal((1, 32, 32, 16))
output = pool_layer(input_feature)
print(f"Input: {input_feature.shape} → After MaxPool: {output.shape}")  # (1, 16, 16, 16)
🔄 3. Data Augmentation – Synthetic Data Generation
🎲 Random transformations during training
📊 Common augmentations

Rotation, width/height shift, zoom, flip, brightness adjustment.

🧠 Why it works

Makes model invariant to transformations, reduces overfitting, especially with small datasets.

⚙️ Implementation

Use tf.image, ImageDataGenerator, or RandomFlip/RandomRotation layers in Keras.

from tensorflow.keras import layers

# Built-in Keras augmentation layers (efficient on-GPU)
data_augmentation = keras.Sequential([
    layers.RandomFlip("horizontal"),
    layers.RandomRotation(0.1),
    layers.RandomZoom(0.1),
    layers.RandomTranslation(0.1, 0.1),
])

# Apply to a batch
sample_images = tf.random.normal((4, 32, 32, 3))
augmented = data_augmentation(sample_images)
print(f"Original shape: {sample_images.shape} -> Augmented: {augmented.shape}")
💡 Best practice: Apply data augmentation only during training, not validation/testing. Use realistic transformations (don't flip digits "6" to "9" for MNIST).
🚀 CIFAR-10 Image Classification with CNN
📸 10 classes: airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck
📊 Dataset

50k train, 10k test. 32x32 RGB images. Challenging: small, diverse objects.

🎯 Architecture

Conv2D(32) → MaxPool → Conv2D(64) → MaxPool → Flatten → Dense(128) → Dense(10)

⚙️ Training

Adam optimizer, categorical crossentropy, data augmentation, 20 epochs.

import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers
import numpy as np
import matplotlib.pyplot as plt

# Load CIFAR-10
cifar10 = keras.datasets.cifar10
(x_train, y_train), (x_test, y_test) = cifar10.load_data()

# Normalize pixel values
x_train = x_train.astype('float32') / 255.0
x_test = x_test.astype('float32') / 255.0

# Convert labels to categorical (one-hot)
y_train_cat = keras.utils.to_categorical(y_train, 10)
y_test_cat = keras.utils.to_categorical(y_test, 10)

print(f"Train: {x_train.shape}, Test: {x_test.shape}")
class_names = ['airplane', 'automobile', 'bird', 'cat', 'deer', 'dog', 'frog', 'horse', 'ship', 'truck']

# Data augmentation for training
data_augmentation = keras.Sequential([
    layers.RandomFlip("horizontal"),
    layers.RandomRotation(0.1),
    layers.RandomZoom(0.1),
    layers.RandomTranslation(0.1, 0.1),
])

# Build CNN Model
def create_cnn_model():
    inputs = keras.Input(shape=(32, 32, 3))
    # Data augmentation (only during training)
    x = data_augmentation(inputs)
    
    # First Conv Block
    x = layers.Conv2D(32, (3,3), activation='relu', padding='same')(x)
    x = layers.BatchNormalization()(x)
    x = layers.Conv2D(32, (3,3), activation='relu', padding='same')(x)
    x = layers.MaxPooling2D((2,2))(x)
    x = layers.Dropout(0.2)(x)
    
    # Second Conv Block
    x = layers.Conv2D(64, (3,3), activation='relu', padding='same')(x)
    x = layers.BatchNormalization()(x)
    x = layers.Conv2D(64, (3,3), activation='relu', padding='same')(x)
    x = layers.MaxPooling2D((2,2))(x)
    x = layers.Dropout(0.3)(x)
    
    # Third Conv Block
    x = layers.Conv2D(128, (3,3), activation='relu', padding='same')(x)
    x = layers.MaxPooling2D((2,2))(x)
    x = layers.Dropout(0.4)(x)
    
    # Dense layers
    x = layers.Flatten()(x)
    x = layers.Dense(128, activation='relu')(x)
    x = layers.Dropout(0.5)(x)
    outputs = layers.Dense(10, activation='softmax')(x)
    
    model = keras.Model(inputs, outputs)
    return model

model = create_cnn_model()
model.compile(optimizer='adam',
              loss='categorical_crossentropy',
              metrics=['accuracy'])

model.summary()

# Train
history = model.fit(x_train, y_train_cat,
                    batch_size=64,
                    epochs=20,
                    validation_data=(x_test, y_test_cat),
                    verbose=1)

# Evaluate
test_loss, test_acc = model.evaluate(x_test, y_test_cat, verbose=0)
print(f"\n✅ Test Accuracy: {test_acc:.4f}")

# Plot training curves
plt.figure(figsize=(12,4))
plt.subplot(1,2,1)
plt.plot(history.history['accuracy'], label='Train Acc')
plt.plot(history.history['val_accuracy'], label='Val Acc')
plt.xlabel('Epoch'); plt.ylabel('Accuracy'); plt.legend(); plt.title('Model Accuracy')
plt.subplot(1,2,2)
plt.plot(history.history['loss'], label='Train Loss')
plt.plot(history.history['val_loss'], label='Val Loss')
plt.xlabel('Epoch'); plt.ylabel('Loss'); plt.legend(); plt.title('Model Loss')
plt.tight_layout()
plt.show()
📊 Expected results: Simple CNN on CIFAR-10 ~70-75% accuracy. With deeper architecture + augmentation + batch norm → 80-85%. Data augmentation typically improves validation accuracy by 3-5%.

🎯 CNN Best Practices & Tips
📐 Architecture

Start with small filters (3x3). Double filters each layer (32→64→128). End with GlobalAveragePooling or Flatten.

🛡️ Regularization

Dropout (0.2-0.5), BatchNormalization, Data Augmentation, L2 weight decay.

⚡ Training

Use learning rate scheduling, early stopping, model checkpointing.

# Advanced: Learning rate scheduling
initial_learning_rate = 0.001
lr_schedule = keras.optimizers.schedules.ExponentialDecay(
    initial_learning_rate, decay_steps=10000, decay_rate=0.9, staircase=True
)
optimizer = keras.optimizers.Adam(learning_rate=lr_schedule)

# Early stopping callback
early_stop = keras.callbacks.EarlyStopping(monitor='val_loss', patience=5, restore_best_weights=True)

# Train with callbacks
model.compile(optimizer=optimizer, loss='categorical_crossentropy', metrics=['accuracy'])
history = model.fit(x_train, y_train_cat, epochs=50, validation_data=(x_test, y_test_cat),
                    callbacks=[early_stop], verbose=1)
