🔄
Recurrent

RNN

What? Neural network with loops: hidden state persists across time steps.

Why? Processes sequences of variable length (text, audio, time series).

Where? Language modeling, speech recognition, time series forecasting.

🧠
Memory Cell

LSTM

What? Gated RNN with forget, input, output gates – controls information flow.

Why? Solves vanishing gradient problem, remembers long-term dependencies.

Where? Stock prediction, text generation, machine translation.

⚠️
Challenge

Vanishing Gradient

What? Gradients become exponentially small in deep RNNs.

Why? Standard RNNs can't learn long-range dependencies (>10 steps).

Where? Solved by LSTMs/GRUs with gating mechanisms.

⚠️ 1. Vanishing Gradient Problem – Why RNNs Failed Initially
📉 Gradients shrink exponentially during backpropagation through time
📐 The problem

For standard RNN, hidden state gradient is product of many weight matrices. If eigenvalues < 1, gradient → 0.

🧠 Consequence

Model can't learn dependencies beyond ~10 time steps. Early words in sentence don't affect later predictions.

⚙️ Solutions

LSTM/GRU (gates), gradient clipping, ReLU activation, residual connections.

📌 Standard RNN: hₜ = tanh(Wₕₕ·hₜ₋₁ + Wₓₕ·xₜ + b) → ∂hₜ/∂hₜ₋₁ = Wₕₕᵀ diag(tanh')
💡 Key insight: Vanishing gradient is why you need LSTMs for stock prediction – stock prices depend on patterns from weeks or months ago.
🧠 2. LSTM – Long Short-Term Memory
🚪 Three gates: Forget, Input, Output – controlled memory cell
🔐 Forget Gate

σ(W_f·[hₜ₋₁, xₜ] + b_f) – decides what to discard from cell state.

📝 Input Gate

σ(W_i·[hₜ₋₁, xₜ] + b_i) – decides what new info to store.

📤 Output Gate

σ(W_o·[hₜ₋₁, xₜ] + b_o) – decides what to output based on cell state.

LSTM gates control information flow: forget irrelevant, store important, output based on memory

📊 3. Time Series Data Preparation for LSTMs
🔄 Sliding window: Create sequences from past data
📐 Window approach

Use last N days to predict next day. X = [t-N...t-1], y = [t].

🎯 Why normalization?

Scale prices to [0,1] or standardize for faster convergence.

⚙️ Train/val/test split

Chronological order! Never shuffle time series.

import numpy as np
import pandas as pd
from sklearn.preprocessing import MinMaxScaler

def create_sequences(data, window_size):
    X, y = [], []
    for i in range(len(data) - window_size):
        X.append(data[i:i+window_size])
        y.append(data[i+window_size])
    return np.array(X), np.array(y)

# Example with synthetic prices
prices = np.sin(np.linspace(0, 100, 500)) + np.random.randn(500)*0.1
scaler = MinMaxScaler()
prices_scaled = scaler.fit_transform(prices.reshape(-1,1))

window = 60  # use 60 days of history
X, y = create_sequences(prices_scaled, window)
print(f"X shape: {X.shape}, y shape: {y.shape}")  # (440, 60, 1), (440,)
📌 Critical: Split time series chronologically: train first 70%, validate next 15%, test last 15%. No random shuffle!
📈 Stock Price Prediction – Complete LSTM Model
🏦 Predict next day's closing price using historical data (60-day window)
📊 Dataset

Historical stock prices (simulated/real). Features: Close price (univariate).

🎯 Architecture

LSTM(50, return_sequences=True) → LSTM(50) → Dense(25) → Dense(1).

⚙️ Training

Adam optimizer, MSE loss, early stopping, 50 epochs.

import tensorflow as tf
from tensorflow.keras import layers, models, callbacks
import numpy as np
import matplotlib.pyplot as plt

# Generate realistic stock-like data (trend + seasonality + noise)
np.random.seed(42)
days = 2000
trend = np.linspace(100, 150, days)
seasonal = 10 * np.sin(2 * np.pi * np.arange(days) / 50)
noise = np.random.randn(days) * 3
prices = trend + seasonal + noise
prices = prices + 50  # base price

# Normalize
scaler = MinMaxScaler()
prices_scaled = scaler.fit_transform(prices.reshape(-1, 1)).flatten()

# Create sequences
window_size = 60
X, y = [], []
for i in range(len(prices_scaled) - window_size):
    X.append(prices_scaled[i:i+window_size])
    y.append(prices_scaled[i+window_size])
X = np.array(X).reshape(-1, window_size, 1)
y = np.array(y)

# Chronological split (70% train, 15% val, 15% test)
train_size = int(len(X) * 0.7)
val_size = int(len(X) * 0.15)
X_train = X[:train_size]
y_train = y[:train_size]
X_val = X[train_size:train_size+val_size]
y_val = y[train_size:train_size+val_size]
X_test = X[train_size+val_size:]
y_test = y[train_size+val_size:]

print(f"Train: {X_train.shape}, Val: {X_val.shape}, Test: {X_test.shape}")

# Build LSTM Model
model = models.Sequential([
    layers.LSTM(50, return_sequences=True, input_shape=(window_size, 1)),
    layers.Dropout(0.2),
    layers.LSTM(50, return_sequences=False),
    layers.Dropout(0.2),
    layers.Dense(25, activation='relu'),
    layers.Dense(1)
])

model.compile(optimizer='adam', loss='mse', metrics=['mae'])
model.summary()

# Early stopping
early_stop = callbacks.EarlyStopping(monitor='val_loss', patience=10, restore_best_weights=True)

# Train
history = model.fit(X_train, y_train,
                    validation_data=(X_val, y_val),
                    epochs=50,
                    batch_size=32,
                    callbacks=[early_stop],
                    verbose=1)

# Evaluate
test_loss, test_mae = model.evaluate(X_test, y_test, verbose=0)
print(f"\n✅ Test MAE: {test_mae:.4f} (scaled)")
print(f"   Actual price MAE: ~${test_mae * (prices.max() - prices.min()):.2f}")

# Plot predictions
y_pred = model.predict(X_test)
y_pred_inv = scaler.inverse_transform(y_pred)
y_test_inv = scaler.inverse_transform(y_test.reshape(-1, 1))

plt.figure(figsize=(14, 6))
plt.plot(y_test_inv, label='Actual Prices', color='blue', alpha=0.7)
plt.plot(y_pred_inv, label='LSTM Predictions', color='red', alpha=0.7)
plt.xlabel('Time Step')
plt.ylabel('Stock Price ($)')
plt.title('LSTM Stock Price Prediction - Test Set')
plt.legend()
plt.grid(alpha=0.3)
plt.show()

# Training curves
plt.figure(figsize=(12,4))
plt.subplot(1,2,1)
plt.plot(history.history['loss'], label='Train Loss')
plt.plot(history.history['val_loss'], label='Val Loss')
plt.xlabel('Epoch'); plt.ylabel('MSE'); plt.legend(); plt.title('Loss Curves')
plt.subplot(1,2,2)
plt.plot(history.history['mae'], label='Train MAE')
plt.plot(history.history['val_mae'], label='Val MAE')
plt.xlabel('Epoch'); plt.ylabel('MAE'); plt.legend(); plt.title('MAE Curves')
plt.tight_layout()
plt.show()
📊 Expected results: LSTM captures trends and seasonality. Test MAE typically 2-5% of price range. On real stock data, LSTMs often outperform ARIMA for non-linear patterns.
🔮 Multi-Step Forecasting – Predict Next 30 Days
📐 Recursive method

Use predictions as inputs for next time steps. Can accumulate error.

🎯 Direct method

Train model to predict multiple steps at once (output shape = horizon).

def predict_future(model, last_sequence, steps, scaler):
    future = []
    current_seq = last_sequence.copy()
    for _ in range(steps):
        pred = model.predict(current_seq.reshape(1, window_size, 1), verbose=0)[0,0]
        future.append(pred)
        current_seq = np.append(current_seq[1:], pred)
    return scaler.inverse_transform(np.array(future).reshape(-1, 1))

# Predict next 30 days
last_60 = X_test[-1]  # last sequence from test set
future_prices = predict_future(model, last_60, 30, scaler)

plt.figure(figsize=(10,5))
plt.plot(range(30), future_prices, marker='o', label='Predicted Future Prices')
plt.xlabel('Days ahead'); plt.ylabel('Price ($)'); plt.title('30-Day Stock Price Forecast')
plt.legend(); plt.grid(alpha=0.3)
plt.show()
