Week 1: Foundations & Classical ML
Day 1 – Python for ML Basics
Topics: NumPy, Pandas, Matplotlib

Example: Load a dataset (iris), compute mean/std, plot scatter matrix.

Day 2 – Data Preprocessing
Topics: Missing values, scaling, encoding

Example: Handle missing data in Titanic dataset, scale features with StandardScaler.

Day 3 – Linear Regression
Topics: MSE, gradient descent, assumptions

Example: Predict house prices (Boston dataset) with LinearRegression from Scikit-learn.

Day 4 – Logistic Regression & Classification Metrics
Topics: Sigmoid, confusion matrix, ROC-AUC

Example: Breast cancer classification; plot ROC curve.

Day 5 – Decision Trees & Random Forest
Topics: Gini impurity, feature importance, bagging

Example: Predict customer churn; compare tree vs forest accuracy.

Day 6 – SVM & K-Nearest Neighbors
Topics: Margin, kernels, distance metrics

Example: Digit recognition (MNIST small) using SVC and KNeighborsClassifier.

Day 7 – Unsupervised Learning (Clustering & PCA)
Topics: K-Means, elbow method, dimensionality reduction

Example: Customer segmentation on mall dataset; visualize clusters.

Week 2: Advanced ML & Deep Learning
Day 8 – Ensemble Methods (XGBoost, Voting)
Topics: Boosting, stacking, hyperparameter tuning

Example: Predict credit card fraud; use GridSearchCV with XGBoost.

Day 9 – Neural Networks with TensorFlow/Keras
Topics: Dense layers, activation functions, backpropagation

Example: Fashion MNIST classification with a 2-layer MLP.

Day 10 – CNNs for Image Data
Topics: Convolution, pooling, data augmentation

Example: CIFAR-10 classifier with Conv2D layers.

Day 11 – RNNs for Sequence Data
Topics: LSTMs, vanishing gradient, time series

Example: Stock price prediction using LSTM (historical prices).

Day 12 – NLP Basics (Text Classification)
Topics: Bag-of-words, TF-IDF, tokenization

Example: Spam detection (SMS data) using Naive Bayes.

Day 13 – Model Deployment (FastAPI + Docker)
Topics: Serialization (pickle/joblib), REST API

Example: Serve a trained iris classifier via local API endpoint.

Day 14 – End-to-End Project
Topics: Combine all steps: data cleaning → modeling → evaluation → deployment

Example: Build a loan approval predictor with web UI using Streamlit.

Bonus Each Day
5-min theory video + 30-min coding practice.

Use Kaggle datasets for real-world feel.

At the end of week 2, you’ll have 4–5 mini-projects and 1 complete full-stack ML project.
